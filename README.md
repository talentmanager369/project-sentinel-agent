# My Antigravity Portfolio

## Welcome to My Zero-Gravity Web Experience

This is an interactive portfolio website featuring a unique **antigravity physics engine** where content cards float freely on the screen. Everything is draggable, and cards maintain momentum when tossed.

## Features

- **Interactive Physics Engine**: Cards drift and bounce realistically
- **Drag & Drop**: Click and drag any card to toss it around
- **Responsive Design**: Works on desktop and mobile devices
- **Modern Aesthetic**: Dark theme with cyan accent colors
- **Smooth Animations**: Blur effects and hover states

## Tech Stack

- HTML5
- CSS3 (with backdrop-filter and custom properties)
- Vanilla JavaScript (Physics simulation with requestAnimationFrame)
- No external libraries required

## How It Works

1. **Drift & Float**: Cards slowly drift due to subtle velocity and friction
2. **Boundary Detection**: Cards bounce off screen edges
3. **User Interaction**: Drag cards with your mouse to apply velocity
4. **Momentum**: Released cards continue moving with calculated velocity
5. **Cosmic Nudges**: Subtle random forces keep cards in motion

## Customization

Edit the following to customize:

- **Colors**: Modify `:root` CSS variables
- **Card Positions**: Update `top`, `left`, `right`, `bottom` properties
- **Card Size**: Adjust `width` values in CSS
- **Physics**: Tweak friction (0.98), velocity clamps (-15, 15), and drift forces
